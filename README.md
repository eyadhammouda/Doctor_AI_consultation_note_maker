# Doctor AI Consultation

I built this mainly to learn about deployment with vercel its a doctor AI consultation note maker application using Next.js, Clerk for authentication & payment, and OpenRouter for AI capabilities using a free gemma model. Front end is a bit crappy as my main goal was learning about deployment and auth

You can have a look at it and play around here: https://saas-1l29uz3ch-eyadsahers-projects.vercel.app/
